DNS_PROBE_FINISHED_NXDOMAIN Hatası: Çözüm Rehberi

Chrome veya Edge'de bir web sitesine girmeye çalışırken "DNS_PROBE_FINISHED_NXDOMAIN" veya "Bu site ulaşılamıyor" hatasıyla karşılaşıyorsunuz. Bu hata genellikle DNS çözümlemesiyle ilgilidir ve çoğunlukla kolayca giderilir.

Bu Hata Ne Anlama Gelir?

NXDOMAIN (Non-Existent Domain), DNS sunucusunun girdiğiniz alan adını çözümleyemediği anlamına gelir. Olası nedenler:

  • Alan adı gerçekten mevcut değil veya silinmiş
  • DNS sunucusu yanıt vermiyor veya yanlış yapılandırılmış
  • DNS önbelleği bozuk veya eski kayıt içeriyor
  • Hosts dosyasında yanlış kayıt var
  • İnternet bağlantısı yok veya kesik

Adım 1: Başka Bir Siteyi Deneyin

Yalnızca tek bir site açılmıyorsa sorun o sitededir. Birden fazla site açılmıyorsa DNS veya bağlantı sorunu sizin tarafınızdadır.

Adım 2: DNS Önbelleğini Temizleyin

1 CMD'yi yönetici olarak açın.
2 ipconfig /flushdns yazın → Enter.
3 "DNS Çözümleyici Önbelleği başarıyla temizlendi" mesajını görmelisiniz.
4 Tarayıcıyı kapatıp açın ve tekrar deneyin.

Adım 3: Chrome DNS Önbelleğini Temizleyin

Chrome kendi DNS önbelleğini tutar. Temizlemek için:

  1. Chrome adres çubuğuna chrome://net-internals/#dns yazın
  2. Clear host cache butonuna tıklayın
  3. Aynı sayfada Sockets sekmesine geçin → Flush socket pools

Adım 4: DNS Sunucusunu Değiştirin

ISS'nin DNS sunucusu sorunluysa Google veya Cloudflare DNS kullanın:

  1. Denetim Masası → Ağ Bağlantıları → WiFi veya Ethernet → Özellikler
  2. İnternet Protokolü Sürüm 4 (TCP/IPv4) → Özellikler
  3. Aşağıdaki DNS sunucu adreslerini kullan seçin
  4. Tercih edilen: 8.8.8.8 — Alternatif: 8.8.4.4
  5. Tamam → kapat

Adım 5: Hosts Dosyasını Kontrol Edin

Hosts dosyasında hatalı bir kayıt DNS çözümlemesini geçersiz kılabilir:

  1. Not Defteri'ni yönetici olarak açın
  2. Dosya → Aç → C:\Windows\System32\drivers\etc\hosts
  3. # ile başlamayan satırları inceleyin — şüpheli kaydı silin
  4. Kaydedin

Adım 6: IP ve DNS'i Yenileyin

CMD'de sırasıyla:

  • ipconfig /release
  • ipconfig /renew
  • ipconfig /flushdns
  • netsh winsock reset

Bilgisayarı yeniden başlatın.

Adım 7: Modemi Yeniden Başlatın

Modemin kendisi de DNS önbelleği tutar. 30 saniye fişini çekip takın.

⚠️ Güvenlik yazılımları veya ebeveyn denetimi uygulamaları bazı siteleri DNS düzeyinde engelleyebilir. Bu uygulamaları geçici olarak devre dışı bırakıp test edin.
