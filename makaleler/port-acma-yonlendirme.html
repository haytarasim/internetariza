Port Açma (Port Forwarding) Nasıl Yapılır?

Online oyun sunucusu kurmak, uzaktan masaüstü erişimi, NAS veya güvenlik kamerası için dışarıdan erişim sağlamak istiyorsanız modemde port yönlendirme (port forwarding) yapmanız gerekir.

Port Forwarding Nedir?

Modeminiz tüm gelen internet trafiğini varsayılan olarak engeller — bu güvenlik açısından iyidir. Port forwarding, belirli bir port numarasına gelen trafiği yerel ağınızdaki belirli bir cihaza yönlendirmenizi sağlar. Örneğin dışarıdan 25565 portuna gelen trafiği Minecraft sunucunuzun kurulu olduğu bilgisayara yönlendirebilirsiniz.

Adım 1: Cihaza Sabit IP Verin

Port forwarding kuralı bir IP adresine bağlanır. Cihazın IP'si değişirse kural çalışmaz. İki yöntem:

  • Modemde DHCP rezervasyonu: Cihazın MAC adresine her zaman aynı IP verilir (önerilen yöntem)
  • Cihazda statik IP: Cihazın IP ayarını manuel yapın (192.168.1.x aralığında, DHCP havuzu dışında)

Adım 2: Modem Paneline Girin

1 Tarayıcıda 192.168.1.1 adresine gidin.
2 Kullanıcı adı ve şifrenizle giriş yapın (varsayılan: admin/admin veya etikette yazar).

Adım 3: Port Forwarding Kuralını Ekleyin

Modem markasına göre menü adı değişir:

  • TP-Link: Gelişmiş → NAT İletme → Sanal Sunucular
  • ZTE: Uygulama → Port Yönlendirme
  • Huawei: Gelişmiş → NAT → Port Eşleme
  • D-Link: Gelişmiş → Port İletme
1 Servis Portu / Dış Port: Dışarıdan gelecek port numarasını girin (örn. 25565).
2 Dahili Port / İç Port: Genellikle aynı numara (25565).
3 Dahili IP / Hedef IP: Cihazın yerel IP adresi (örn. 192.168.1.100).
4 Protokol: TCP, UDP veya her ikisi (uygulamanın belgelerine bakın).
5 Kaydedin ve modemi yeniden başlatın.

Popüler Uygulamalar için Port Listesi

Uygulama Port Protokol
Minecraft Sunucu 25565 TCP+UDP
Uzak Masaüstü (RDP) 3389 TCP
Web Sunucusu (HTTP) 80 TCP
Web Sunucusu (HTTPS) 443 TCP
FTP 21 TCP
SSH 22 TCP
CCTV / IP Kamera 8080 veya üretici bazlı TCP

Adım 4: Port Açılışını Test Edin

Canlı IP adresinizi öğrenmek için bu rehbere bakın. Ardından canyouseeme.org sitesinde port numaranızı girerek dışarıdan erişilebilir olup olmadığını test edin.

⚠️ Güvenlik uyarısı: Port açmak güvenlik riskini artırır. Yalnızca ihtiyaç duyduğunuz portları açın. RDP (3389) gibi kritik portları açıyorsanız güçlü şifre kullanın ve mümkünse port numarasını değiştirin.

CGN / Carrier-Grade NAT Sorunu

Bazı ISS'ler abonelere gerçek bir genel IP adresi vermez, paylaşımlı IP kullanır (CGN/CGNAT). Bu durumda port forwarding çalışmaz. ISS'nizden statik IP veya CGN'den çıkış talep edebilirsiniz (ücretli olabilir).
