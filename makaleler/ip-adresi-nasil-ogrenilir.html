IP Adresim Nedir? Yerel ve Genel IP Nasıl Bulunur?

IP adresi sorun giderme, port açma ve ağ yapılandırması için sıkça ihtiyaç duyulan bir bilgidir. Yerel IP ile genel IP birbirinden farklıdır — ikisini de nasıl bulacağınızı öğrenin.

Yerel IP ve Genel IP Farkı

Yerel IP (Local IP): Modeminizin ev ağınızdaki cihazlara atadığı adrestir. Genellikle 192.168.1.x veya 192.168.0.x formatındadır. Yalnızca ev ağınız içinde geçerlidir.

Genel IP (Public IP): İnternet sağlayıcınızın modeminize atadığı adrestir. Dış dünyadan sizi tanımlayan IP budur. Port forwarding, uzaktan erişim ve coğrafi hizmetler için önemlidir.

💡 Evdeki tüm cihazlar aynı genel IP'yi paylaşır. Her birinin farklı bir yerel IP'si vardır.

Genel IP Adresinizi Öğrenin

En hızlı yol: herhangi bir tarayıcıda Google'a "ip adresim nedir" yazın. Google anında genel IP adresinizi gösterir.

Alternatif siteler:

  • whatismyip.com
  • ipinfo.io
  • myip.com

Windows'ta Yerel IP Öğrenme

Yöntem 1: ipconfig Komutu

1 Win+R tuşlarına basın → cmd yazın → Enter.
2 ipconfig yazın → Enter.
3 Ethernet veya Kablosuz LAN bölümündeki IPv4 Adresi satırını okuyun.

Yöntem 2: Ağ Ayarları

  1. Görev çubuğundaki WiFi veya Ethernet simgesine tıklayın
  2. Ağ ve İnternet ayarları → bağlı ağın özelliklerine girin
  3. IPv4 adresi satırında yerel IP görünür

Mac'te Yerel IP Öğrenme

  1. Elma menüsü → Sistem Tercihleri → Ağ
  2. Sol panelde aktif bağlantıyı seçin (WiFi veya Ethernet)
  3. Sağ panelde IP adresi görünür

Terminal ile: ifconfig | grep "inet "

Android'de Yerel IP Öğrenme

  1. Ayarlar → WiFi
  2. Bağlı olduğunuz ağa dokunun veya dişli simgesine basın
  3. IP adresi satırında yerel IP görünür
💡 Android 10+ sürümlerde Gelişmiş seçeneklerde görebilirsiniz. Göremiyorsanız IP ayarı'nı Statik'e çekince tüm ağ bilgileri açılır — not alın, sonra tekrar Dinamik'e alın.

iPhone / iPad'de Yerel IP Öğrenme

  1. Ayarlar → WiFi
  2. Bağlı ağın yanındaki (i) simgesine dokunun
  3. IP Adresi satırında yerel IP görünür

Modem Arayüzünden IP Bilgileri

Modem panelinde (192.168.1.1) Durum → WAN Durumu bölümünden hem genel IP adresinizi hem de bağlantı bilgilerinizi görebilirsiniz.

Statik mi, Dinamik mi?

Genel IP adresiniz büyük ihtimalle dinamiktir — modem her yeniden başladığında değişebilir. Uzaktan erişim veya sunucu çalıştırmak için statik IP almak isterseniz ISS'nize başvurabilirsiniz (ücretli hizmet).
